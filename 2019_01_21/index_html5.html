html5의 symantec tag
html5이전에는 구역을 나누어 사용할 경우 div tag을 이용해서 설정하고 id나 class를 부여해서 꾸미기를 했다.
html5부터는 tag을 의미 있는 단어를 사용해서 설정을 하고 id나 class 부여해서 모양꾸미기를 한다
  • header : 본문의 제목을 표시하는 영역
  • section : 본문의 큰 영역
  • article : 본문의 문단 영역
  • footer : 본문의 꼬리말
