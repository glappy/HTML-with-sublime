class 속성

tag들을 그룹으로 묶어서 공통된 속성을 적용하는 방법

여러 tag에 같은 이름의 class를 지정하고 style에 속성을 설정하면 공통된 모양을 만들 수 있다.

style sheet에 지정하는 방법

tag 이름

tag 이름에 속성을 설정하면 html 문서 내의 모든 tag에 같은 모양이 적용된다

#id

tag에 id 속성을 지정하면 특정한 한개의 tag에만 모양을 지정할 수 있다.

id 속성은 pk성질을 갖고 있어서 하나의 html 문서 내에서는 중복하여 같은 이름을 지정하면 안된다.

.class

tag에 class 속성을 지정하면 여러 tag에 공통된 속성을 지정할 수 있다.

종류가 다른 tag에 같은 이름의 class를 지정할 경우 tag.class형식으로 지정하면 특정 tag에 설정된 class들만 모양을 지정할 수 있다.
